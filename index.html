Intro


A bookmarklet is a bookmark stored in a web browser containing JavaScript commands which add new features to the browser.

This page holds a list of SEO-specific bookmarklets I made to speed up common SEO tasks.

To use one of these bookmarklets:

  • Drag and drop one of the links onto your bookmark bar
  • Click the bookmarklet on your bookmark bar to use it

You can also put the bookmarklets in a folder to prevent them from cluttering your bookmark bar.





On-page SEO


Meta Title

Displays the meta title for the page you're on in an alert box.

Meta Description

Displays the meta description for the page you're on in an alert box.

H1

Displays the H1 for the page you're on in an alert box.

Canonical

Displays the canonical URL for the page you're on in an alert box.

Robots.txt

Takes you to the Robots.txt file of the site you're on.

SEO Report

Produce an on-page SEO report for the URL you're on.





KW Research


KW - Ahrefs

Get Ahrefs keyword data by inputting a target keyword in the prompt.

KW - SEMrush

Get SEMrush keyword data by inputting a target keyword in the prompt.

KW - Moz

Get Moz keyword data by inputting a target keyword in the prompt.

KW - Google Trends

Get Google Trends keyword data by inputting a target keyword in the prompt.

KW - All 4

Get keyword data for all 4 above tools by inputting a target keyword in the prompt.

You will have to enable popups for this to work.





Backlinks


Backlinks - Current URL

Takes you to the Ahrefs backlinks report for the URL you're currently on.

Backlinks - Entire Domain

Takes you to the Ahrefs backlinks report for the entire domain you're currently on.





Featured Snippets


Featured Snippets - Current URL

See all Featured Snippets for the current URL you're on.

Featured Snippets - Entire Domain

See all Featured Snippets for the entire domain you're on.





Validation Tools


AMP Validator

Use on a non-amp page to check if the AMP variant is valid.

Schema Validator

Runs the current URL through Google's Schema / Structured Data Testing Tool.

Mobile Friendly Test

Runs the current URL through Google's Mobile-Friendly Test.






PageSpeed


PageSpeed - Current URL

Runs the current page you're on through Google's PageSpeed Insights tool.
